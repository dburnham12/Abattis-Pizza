# Abattis Pizza

This project is a simple, responsive website for Abattis Pizza built using vanilla HTML, CSS, and JavaScript to showcase the restaurant’s Italian-inspired brand, mission, and operating hours. It highlights the shop’s commitment to high-quality dining at friendly prices while making it easy for customers to learn about and place orders at any time of day.
